每个 Vue 组件实例在创建时都需要经历一系列的初始化步骤，比如设置好数据侦听，编译模板，挂载实例到 DOM，以及在数据改变时更新 DOM。

![Vue LiefCycle]({{ "images/vue_lifecycle.png" | relative_url }})

组合式 API 与 声明式 API 钩子命名还有些许区别：

| 声明式 API   | 组合式 API    |
| :----------- | :------------ |
| beforeCreate | setup         |
| created      | setup         |
| beforeMount  | onBeforeMount |
| mounted      | onMounted     |
