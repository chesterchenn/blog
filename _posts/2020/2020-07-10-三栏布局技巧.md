三栏布局：两边定宽，中间自适应

## BFC
左右设置 float，中间设置 overflow 属性触发 BFC  
```
核心代码：
<div class="left"></div>
<div class="right"></div>
<div class="main"></div>

.left {
  float: left;
}
.right {
  float: right;
}
.main {
  overflow: auto;
}
```
