### CSS 标准单位
#### 绝对单位长度  

| 单位 | 名称 |
| --- | --- |
| px  | Pixels 像素 |

#### 相对单位长度

| 单位 | 描述 |
| --- | --- |
| em  | 如果是字体大小，则相对于父元素字体大小，如果是宽度等其他属性，则相对于元素本身字体大小 |
| rem | 相对于根元素的字体大小 |
| vw  | 相对于视窗宽度的1% |
| vh  | 相对于视窗高度的1% |

注：rpx 是微信小程序里面 WXSS 中的单位
