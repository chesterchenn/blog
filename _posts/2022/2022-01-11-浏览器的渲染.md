```plain
                                                               ┌───┐
┌──────────────────────────────────────────────────────────┐   │ D │
│                    User Interface                        ├──>│ a │

                           │                             │     │ a │
                           │                             │     │   │
                           ▼                             │     │ P │
┌──────────────────────────────────────────────────────┐ │     │ e │
│                    Browser Engine                    ├─┼────>│ r │
└──────────────────────────┬───────────────────────────┘ │     │ s │
                           │                             │     │ i │
                           │                             │     │ s │
                           ▼                             │     │ t │
┌──────────────────────────────────────────────────────┐ │     │ e │
│                   Rendering Engine                   │ │     │ n │
└────┬──────────────────┬──────────────────┬─────────┬─┘ │     │ c │
     │                  │                  │         │   │     │ e │
     │                  │                  │         │   │     └───┘
     ▼                  ▼                  ▼         ▼   ▼
```
