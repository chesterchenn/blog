- 选择排序是一种不稳定的算法。
- 选择排序的时间复杂度为 O(n<sup>2</sup>)。
- 选择排序的空间复杂度为 O(1)。

选择排序的伪代码：
```
i ← 0
while i < length(A)
  minIndex ← i
  j ← i + 1
  while j < length(A)

For i = 0 to length(A)
```
