# 闭包
Created: 2019-04-24  
Last Modified: 2019-04-24  

闭包(closure), 在MDN上定义：A closure is the combination of a function and the lexical environment within which that function was declared。(闭包是一个函数以及声明该函数的词法环境的组合)。  

一个闭包必备的条件：
- 一个函数
- 该函数提供了使外部访问函数内部变量的返回
