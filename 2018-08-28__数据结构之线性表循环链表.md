# 循环链表( Circular Linked List )
Created: 2018-08-28  
Last Modified: 2018-08-28  

循环链表就是尾节点的指针不指向NULL，而是指向头结点，从而形成一个循环链表。相对的，如果是循环双向链表，头结点的前置指针也指向尾结点。循环链表的操作跟单链表与双向链表的基本操作大致相同。  

```
//===============
// 遍历循环链表
//===============
void traversal(struct Node *node) {
  struct Node *temp = first;
  while(temp != first) {
    // do something
    temp = temp->next;
  }
}
```
