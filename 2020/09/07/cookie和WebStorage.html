cookie

  1. cookie зҡ„еӨ§е°Ҹиў«йҷҗеҲ¶еңЁ 4KBгҖӮ
  2. cookie дјҡйҡҸзқҖ HTTP дәӢеҠЎдёҖиө·еҸ‘йҖҒпјҢеӣ жӯӨдјҡжөӘиҙ№дёҖе°ҸйғЁеҲҶеёҰе®ҪгҖӮ
  3. иҰҒж“ҚдҪң cookie жҜ”иҫғйә»зғҰгҖӮ
  4. cookie йҖҡиҝҮ name=value зҡ„еҪўејҸеӯҳеӮЁгҖӮ
  5. дёҖиҲ¬й»ҳи®ӨжҳҜжңҚеҠЎеҷЁз”ҹжҲҗпјҢеҸҜд»Ҙи®ҫзҪ®еӨұж•Ҳж—¶й—ҙгҖӮеҸҜд»ҘеңЁжөҸи§ҲеҷЁз”ҹжҲҗпјҢй»ҳи®Өе…ій—ӯеӨұж•ҲгҖӮ

sessionStorage дёҺ localStorage

Web Storage еҸҜд»ҘеҲҶдёә sessionStorage е’Ң localStorageгҖӮ

  1. sessionStorage е°Ҷж•°жҚ®дҝқеӯҳеңЁе®ўжҲ·з«Ҝ session еҜ№иұЎдёӯгҖӮlocalStorage е°Ҷж•°жҚ®дҝқеӯҳеңЁе®ўжҲ·з«Ҝжң¬ең°зЎ¬д»¶и®ҫеӨҮдёӯгҖӮ
  2. sessionStorage дёәдёҙж—¶дҝқеӯҳпјҢиҖҢ localStorage дёәж°ёд№…дҝқеӯҳгҖӮ
  3. sessionStorage жҳҜд»…еңЁеҪ“еүҚдјҡиҜқдёӯжңүж•ҲпјҢеңЁеҗҢдёҖдёӘзӘ—еҸЈдёӯеҲ·ж–°жҲ–иҖ…иҝӣе…ҘеҸҰдёҖдёӘеҗҢжәҗйЎөйқўпјҢд№ҹдҫқж—§еӯҳеңЁгҖӮдҪҶжҳҜе…ій—ӯжөҸи§ҲеҷЁзӘ—еҸЈжҲ–иҖ…зӢ¬з«Ӣжү“ејҖж–°йЎөйқўйғҪдјҡдёҚдёҖж ·гҖӮ
  4. sessionStorage е’Ң localStorage еҸӘиғҪеӯҳеӮЁеӯ—з¬ҰдёІзұ»еһӢгҖӮ
  5. WebStorage дёҚдјҡйҡҸзқҖ HTTP header еҸ‘йҖҒеҲ°жңҚеҠЎеҷЁз«ҜпјҢжүҖд»Ҙе®үе…ЁжҖ§зӣёеҜ№дәҺ cookie жқҘиҜҙжҜ”иҫғй«ҳдёҖдәӣпјҢдёҚдјҡжӢ…еҝғжҲӘиҺ·пјҢдҪҶжҳҜд»Қз„¶еӯҳеңЁдјӘйҖ й—®йўҳгҖӮ
