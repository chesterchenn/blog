# 插入排序(Insertion Sort)
Created: 2018-10-31  
Last Modified: 2018-10-31  

![image](./images/insertion_sort.gif)  
图片来源：维基百科

- 插入排序是一种稳定的算法。
- 插入排序的时间复杂度为 O(n<sup>2</sup>)。
- 插入排序的空间复杂度为 O(1)。

插入排序就是构建有序序列，对于未排序的数据，在已排序序列中从后往前扫描，找到合适的位置插入。  
插入排序跟选择排序很相似。

插入排序的伪代码：
```
i ← 1
while i < length(A)
  j ← i
  while j > 0 and A[j-1] > A[j]
    swap A[j] and A[j-1]
    j ← j - 1
  end while
  i ← i + 1
end while
```
